# Evaluation


| 属性 (Attribute) | 评估重点 (Key Assessment Focus) |
| :--- | :--- |
| **香气** (Fragrance/Aroma) | 干粉和湿水后的气味；潜在风味和复杂度的第一印象。 |
| **风味** (Flavour) | 主要口感特征；所有味觉和鼻后嗅觉的综合印象。 |
| **余韵** (Aftertaste) | 吞咽后风味残留的长度和愉悦感；正向余韵需干净、带甜。 |
| **酸质** (Acidity) | 咖啡的活泼度或亮度；高品质酸度需清脆、甜美。 |
| **醇厚度** (Body) | 咖啡的物理口感和重量感（如：轻盈或糖浆状）。 |
| **平衡度** (Balance) | 风味、余韵、酸质、醇厚度等各属性的和谐统一性。 |
| **一致性** (Uniformity) | 多杯（同批次）咖啡之间的风味和品质是否高度一致。 |
| **干净度** (Clean Cup) | 从头到尾没有干扰性的负面风味或瑕疵。 |
| **甜度** (Sweetness) | 咖啡中天然甜感和愉悦的饱满度。 |
| **总体评价** (Overall) | 评测者对咖啡整体品质和价值的个人、全面评价。 |

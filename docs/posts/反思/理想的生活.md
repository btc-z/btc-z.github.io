# 理想的生活

没有照顾自己的灵魂就像是没有照顾树木的根。没有根的树，只是在勉强度日，仅仅是活着罢了。

其实，我能有现在这份工作真的很幸运。不用担心钱，同事们性格好，老板愿意挡下困难，而且也会共情。这也许是我现在可以去思考这个问题的基石吧。如若不然，我甚至不知道要如何开始。我希望以下设想可以尽可能天马行空一些，那些条条框框的限制其实都是层层自己设定的恐惧，社会堆积的恐惧，自我叠加上的恐惧，到头来，有什么是不可能的呢？换个想法，就当作是一场梦里或者另一个平行世界里的我着笔好了。

清晨被阳光照醒，咖啡自然是第一位，早餐是我每天最爱的一顿饭，因为大多数时候都是被饿醒的。其实工作哲学的本身也许是那种迫不及待开始的感觉，而不是啊，我不得不开始的感觉。再或者说工作的哲学是，我今天可以学什么？昨天学的好像有了新的头绪，终于有机会可以试试看了。当然会好，可以有一起钻研的战友，不然有挑战的问题必然都过于孤独。

午饭，我喜欢简洁的吃，但喜欢和一起挑战问题的人聊工作以外的问题。因为这是仅有的可以看到不带着工作面具的他们是谁的机会。那些整天都带着工作面具的人，反倒是让我畏惧，因为我并没有办法看到他们。看不到的，都是黑暗中的危险，本能告诉我。

下午的时间我喜欢留给完成琐碎的工作，当然没有效率的时候，就去休息或者读书，去散步，去思考接下来最重要，最有挑战，最有价值的问题去解决。等待落日的时间其实都是在准备夜晚的过程，也是在准备第二天到来的过程。夜晚总会来，肚子迟早会饿。晚饭想和家人或者朋友一起吃，与其看电视，更喜欢吃饭的时候可以兴高采烈的聊天，也许是因为喜欢部落篝火文化的原因？围着篝火边，可以各自讲自己的幻想，讲自己的恐惧，自己的好奇心，求知欲，和欲望，故事的目的难道不就是一面镜子么？

夜晚是美好的，但是没有繁星的夜晚，没有虫鸣的夜晚，没有歌声的夜晚，没有山谷长啸回响的夜晚，这样的人生值得活么？只有思考没有感受的夜晚，是值得度过的夜晚么？就像是光和影，他们需要彼此共存，又像是树根和树枝。

独处的时间，是留给自己的。肉体的独处时允许精神栖息的土壤。也修养灵魂，才可以与其对话。当我们坠入梦境之时，我们到底在梦什么？那梦醒后，不再记起的一个一个世界，一个一个人，他们是谁，我们和他们之间的故事不是值得期待的么？那是真的只属于自己的世界，一个人的精神世界是无法被玷污的，梦是纯纯粹粹属于你的东西。它是真正的你的房间，它是真正允许你为自己哭泣的地方。
